¿Qué es Global Azure Bootcamp?

Sábado, April 25, 2015   again!                      

En abril de 2013 el primer Global Azure Bootcamp   se celebró en más de 90 ubicaciones de todo el mundo! En marzo de 2014 fue cubierto con 136 ubicaciones!  Este año estamos de nuevo dedicando todo un día para ayudar a miles de personas a acelerar el desarrollo de aplicaciones de Cloud Computing utilizando Azure.

Además de esta gran oportunidad de aprendizaje tendremos sesiones con laboratorios
